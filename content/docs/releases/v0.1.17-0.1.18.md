# v0.1.17 – v0.1.18

Released: February 26 – 27, 2026

---

CI/CLI decoupling block.

- Generated CI workflows are no longer tied to the cli plan that produced them — workflows can run in CI without the cli having to re-plan first.
- Build scope and package build-order fixes so the workspace builds reliably from a clean checkout.
- Templates updated for the new CI/CLI contract.
- `init` no longer runs the cloud-init step automatically — `cloud:init` is a deliberate, separate command users invoke when they're ready to deploy.
