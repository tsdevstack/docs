# v0.1.24 – v0.1.27

Released: March 1 – 5, 2026

---

Cloud SDK migration + Azure provider integration.

## Cloud SDK migration

Replaced shell-outs to the `gcloud`, `aws`, and `az` CLIs with native SDK calls (`@google-cloud/*`, `@aws-sdk/*`, `@azure/arm-*`, `@azure/identity`). The provider CLIs are no longer required on the developer's machine or in CI for generation — only `terraform` is needed for actual deployment.

## Azure provider — full integration

- Azure provider added as a peer of GCP and AWS
- Container Apps + Key Vault + Front Door + Storage wiring
- Managed identity for Key Vault access in Container Apps (no shared secrets in env)
- ACR pull fixes for container deployment
- Azure Terraform provider registration fix
- Custom WAF rules support for Azure Front Door
- Azure CICD workflow generation
- Container performance defaults raised
- Database migration error handling improvements
