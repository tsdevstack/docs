# v0.2.4

Released: April 7, 2026

---

Init-flow + GCP-bootstrap fixes that landed same day as v0.2.3.

- **`init` git-repo handling**: `git init` moved out of `clone-template-repo` (which was producing a nested `.git` dir inside the cloned template) and runs in the project root after the clone instead.
- **Lockfile reconciliation**: `init` now runs `npm install` a second time after the first install to work around an upstream npm bug where the first install can leave the lockfile out of sync with workspace dependencies.
- **GCP `infra:bootstrap`**: 15-second delay inserted between API enablement and IAM binding to avoid a propagation race that intermittently caused IAM operations to fail with "API not yet enabled."
- **`infra:deploy`**: now prints a DNS verification table at the end summarising what to point at where.
- Template `.gitignore` synced (secrets patterns, npmrc, removed outdated Vercel section).
